# Header-Component-Next-js

This repository contains two ready-to-use header components (Light and Dark) built for Next.js applications.  
Both components are fully responsive and styled with **Tailwind CSS**, and they use **lucide-react** for icons.

These headers are designed to help you quickly integrate a modern, clean navigation bar into any Next.js project.
